# OSU Strategic Plan — Strategic Alignment options

This file holds OSU's strategic plan (*Prosperity Widely Shared*) so the skill can
offer authors a **pick-list** for the brief's **Strategic Alignment** section,
instead of asking them to recall a goal from memory. Naming a *specific* plan
commitment is exactly what reviewers look for (see `rubric.md` → Strategic
Alignment), so this turns a commonly-weak section into a quick, accurate choice.

## How the skill uses this file

When working the Strategic Alignment section (Step 3 interview, brief mode):

1. Read the goal/commitment list under **Strategic alignment options** below.
2. Present it to the author as a **numbered pick-list** in a fenced `questions`
   block — one goal per line — and let them answer by number (one or more), e.g.
   "Which of OSU's strategic goals does this most directly advance? (pick any)".
3. Take the author's pick(s) verbatim as the named commitment(s), then ask them
   (in their own words) *how* the work advances it — do **not** invent the
   connection yourself. Write the result into `strategic_alignment` as the named
   goal + the author's one-line rationale.
4. If nothing fits, that's a real signal — capture it plainly rather than forcing
   a goal, and note it for the readiness summary.

Never assert an alignment the author didn't choose, and never reword a goal into
something the plan doesn't say (this is a *facts come from the source* rule, same
as the two hard rules in `SKILL.md`). If this file still only contains the
placeholder below, fall back to the open-ended Strategic alignment question in
`reviewer-questions.md` and tell the author the pick-list isn't loaded yet.
